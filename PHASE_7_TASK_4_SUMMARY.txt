使用方法
================================================================================

【基本的な使い方】
```rust
use nantaraquad::api::*;

// GameEngine作成
let mut engine = GameEngine::new(160, 120, 60);

// 描画
engine.drawing.cls(0);
engine.drawing.rectfill(10, 10, 50, 50, 8); // red
engine.drawing.circle(100, 100, 20, 7);     // white

// 入力処理
engine.input.press_key(Key::Up);
if engine.input.btn(Key::Up) {
    // プレイヤー移動
}
engine.input.update_frame();

// カメラ
engine.camera.follow(player_x, player_y, 0.1);
```

【描画API】
- rect(x, y, w, h, color) - 矩形枠線
- rectfill(x, y, w, h, color) - 矩形塗りつぶし
- circle(x, y, r, color) - 円枠線
- circfill(x, y, r, color) - 円塗りつぶし
- line(x1, y1, x2, y2, color) - ライン
- pset(x, y, color) - ピクセル設定
- pget(x, y) - ピクセル取得
- cls(color) - キャンバスクリア

【入力API】
- btn(Key) - キーが押されているか
- btnp(Key) - キーが今フレーム押されたか
- press_key(Key) / release_key(Key) - キー状態変更
- update_frame() - フレーム更新

【カメラAPI】
- world_to_screen(wx, wy) - ワールド→スクリーン変換
- screen_to_world(sx, sy) - スクリーン→ワールド変換
- follow(target_x, target_y, speed) - スムーズフォロー
- set_zoom(zoom) / zoom_in() / zoom_out() - ズーム制御
